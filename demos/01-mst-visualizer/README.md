MST Visualizer
==============

Interactive step-by-step visualisation of Kruskal's Minimum Spanning Tree algorithm.

Run:
    python main.py

Controls:
    Click on canvas       — add a terminal vertex
    [N]ext Step           — process one edge
    [A]uto Run            — animate automatically
    [R]eset               — reset algorithm, keep vertices
    [C]lear All           — remove all vertices
    [Space]               — toggle auto-run

Requirements:
    pygame >= 2.0
    pip install -r requirements.txt
